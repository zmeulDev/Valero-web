Clone a GitHub Repository

  1. 1. Install Git

    sudo apt install git

  2. 2. Clone Repository to Specific Directory

    cd /var/www/valero
sudo rm -rf public/
git clone https://github.com/zmeulDev/valero.git .
ls -la

  3. 3. Set Proper Permissions

    sudo chown -R $USER:$USER /var/www/valero
sudo chmod -R 755 /var/www/valero
sudo chown -R www-data:www-data /var/www/valero/public
sudo chmod 664 /var/www/valero/public/sitemap.xml
sudo chown -R www-data:www-data /var/www/valero/storage
sudo chown -R www-data:www-data /var/www/valero/bootstrap/cache

Troubleshooting

If you encounter permission or cloning issues:

  • Ensure you have the correct SSH keys or access tokens
  • Check your internet connection
  • Verify repository URL
